## MySQL Connection

```bash
MYSQL_DATABASE = job_interview_db
MYSQL_USER = host
MYSQL_ROOT_PASSWORD = host
MYSQL_PASSWORD = host
MYSQL_HOST = 0.0.0.0

```

## Quickstart

Run project with Makefile

```bash
  make
```
description
-  `cd core_app` and run `yarn build` build and complie core_app (node)
- `docker-compose down`
- `docker-compose up -d`
